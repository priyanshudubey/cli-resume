# Resume CLI
My command line resume

# 📝 Install
```
$ npm i -g priyanshu-resume
```

# 📜 Run 
```
node index
```

# 📜 How it looks 
```
----------------------------------------------------------

----------------------------------------------------------
----------------------------------------------------------

  Website
(Move up and down to reveal more choices)
```
